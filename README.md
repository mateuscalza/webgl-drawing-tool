# WebGL Getting Started

Drawing tool powered by ThreeJS (WebGL), D3 and React. Demo at [http://webgl-react.vercel.app/](http://webgl-react.vercel.app/).

## Run as development

```bash
# Install dependencies
npm install

# Run
npm start
```
